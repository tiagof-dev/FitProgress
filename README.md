FitProgress 💪

FitProgress é um aplicativo PWA para acompanhamento de treinos e evolução física.

Funcionalidades

- 📅 Calendário de treinos
- 💪 Registro de exercícios
- 📊 Dashboard de progresso
- 🔥 Controle de sequência de treinos
- 🎯 Metas mensais
- 📚 Histórico de treinos
- 🌙 Tema escuro e claro
- 📱 Instalação como aplicativo (PWA)

Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- PWA (Progressive Web App)

Instalação

1. Acesse o aplicativo pelo navegador.
2. Toque em "Instalar aplicativo".
3. Utilize normalmente pelo celular ou computador.
